# Saucedemo Automation Project

## Overview
This repository contains UI Automation scripts for saucedemo project.

The automation covers login functionality and the validation of success/failed login.

## Prerequisites
Please ensure the following are installed:
- Katalon Studio (version 10.x.x or latest version is recommended)
- Java (compatible with katalon)
- Chrome browser (latest version)
- Also update webdrivers chrome in Katalon Studio
- Git

## How to setup
1. Clone repository
git clone https://github.com/fiqihz/saucedemo-project.git

2. Open project
- Open Katalon Studio
- Click Open Project
- Select the cloned project folder 

## How to run test
1. Navigate to Test Suites
2. Select TC_Login
3. Click Run using chrome browser

## Test reports
Reports are automatically generated under /Reports folder

## Notes:
All executed test cases refer to the manual test cases documented in Google Docs.
